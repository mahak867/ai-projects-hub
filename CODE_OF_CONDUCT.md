# Code of Conduct

## Our Standard

This project is a learning resource. Everyone — beginner to expert — is welcome.

**Be:**
- Respectful and constructive in issues and PRs
- Patient with beginners (this repo is explicitly for learning)
- Specific when reporting bugs or requesting changes

**Don't:**
- Post dismissive or condescending comments
- Open issues to demand features without context
- Spam or self-promote unrelated content

## Enforcement

Violations → comment removed. Repeated violations → blocked from the repo.

Report issues to the repo owner via GitHub.
