# Contributing to AI Projects Hub

Thank you for considering contributing! Here's how to get involved.

## Adding a New Project

1. Fork the repo
2. Create a folder: `NN-your-project-name/`
3. Include:
   - `README.md` — setup, usage, what you learn
   - Working code files
   - `requirements.txt`
4. Add your project to the main `README.md` table
5. Open a PR with a screenshot or demo GIF

## Project Requirements

Every project must:
- [ ] Run in under 5 minutes
- [ ] Have a clear README with setup steps
- [ ] Use real APIs (not just mock data)
- [ ] Include a "Key concept" section explaining what it teaches
- [ ] Not require paid services beyond what's listed

## Code Standards

- Python: follow PEP 8, add type hints
- Include error handling
- No hardcoded API keys (use env vars)
- Comments for non-obvious code

## Good First Issues

Look for issues tagged `good first issue` — these are smaller improvements like:
- Adding a new feed to the news analyst
- Adding a new document type to the extractor
- Improving error messages
- Adding tests

## Questions?

Open an issue or start a discussion. Happy to help!
